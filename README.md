# Borradores privados de diseño web

Cada carpeta contiene la maqueta de un local. Todas las páginas llevan `noindex`: no aparecen en buscadores y solo las ve quien tiene el enlace.

**Fotos:** colocarlas en `<carpeta>/img/` con estos nombres: `hero.jpg`, `p1.jpg`, `p2.jpg`, `p3.jpg`, `p4.jpg` (JPG, ~1600 px de ancho máximo, <300 KB).

Cualquier maqueta se retira de inmediato a petición del titular del negocio.
